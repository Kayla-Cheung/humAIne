# humAIne

*Where Human Intuition Meets Artificial Intelligence.*

这个仓库（Repository）是用来记录对前沿 AI 技术的去公式化拆解、STS（科学、技术与社会）反思以及人文直觉。

我们拒绝纯粹的数学公式堆砌，拒绝盲从“暴力美学（Scale is all you need）”。我们致力于用哲学、心理学和人类常识去重新解构 AI 架构。
